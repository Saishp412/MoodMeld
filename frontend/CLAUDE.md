# MoodMeld 2.0

## Project Structure
- `frontend/` - Next.js 16 + Tailwind CSS + Framer Motion
- `backend/` - FastAPI + MongoDB Atlas + JWT Auth

## Database
- MongoDB Atlas (cloud)
- Motor async driver

## Running
- Frontend: `cd frontend && npm run dev` (port 3000)
- Backend: `cd backend && uvicorn app.main:app --reload` (port 8000)
